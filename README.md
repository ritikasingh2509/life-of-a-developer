# 🚀 Life of a Developer

An interactive storytelling website that showcases the journey of a developer — from learning basics to debugging chaos and finally achieving success.

## 🌟 Features
- Scroll-based storytelling experience
- Smooth animations using GSAP
- Interactive UI elements
- Background music & particle effects
- Fully responsive design

## 🛠️ Tech Stack
- HTML, CSS, JavaScript
- GSAP (animations)
- ScrollTrigger
