# 🌐 Life of a Developer

An interactive storytelling web experience that captures the journey of a developer—from writing the first line of code to debugging chaos, deadline pressure, and finally achieving success.

## 🚀 Live Demo

👉 https://life-of-a-developer-9g1p.vercel.app/

## 📌 Project Description

Life of a Developer is an interactive storytelling website designed to simulate the emotional and technical phases of a programmer’s journey. Instead of static content, users experience the narrative through smooth scroll-based animations, transitions, and interactive UI elements.

The journey includes:

* 📘 Learning Phase (HTML, CSS, JS)
* 🐞 Debugging Chaos (errors, glitches, frustration)
* ⏳ Deadline Pressure (countdown & urgency)
* ☕ Success Moment (final working state)

Built using HTML, CSS, JavaScript, and GSAP, the project focuses on combining storytelling with modern UI/UX practices. It is fully responsive and optimized for multiple devices.
